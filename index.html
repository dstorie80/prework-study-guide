HTML

  • HEAD
    • Head - Contains the metadata to render the page
    • Title - Defines the title of the webpage (shown in the browser)
    • Style - Defines the CSS styles associated w/ webiste
    • Base - Defines the base URL for a webpage
    • Links - Connects an external resource to the HTML doc
    • Meta - Defines metadata sucha as a character set description, keywords, author and/or viewpoint
    • Script- Embeds client side scripts in the HTML
  • BODY
    • h1 - Headers
      • h1
      • h2
      • h3
    • p - Paragraphs
    • ul - Unordered lists
    • li - List items
    • img - Information about images
      • src - Where the image comes from
      • alt - Gives alternative description of the image
      • width - Image width
      • height - Image height
    • br - Creates a line break between elements
    • a - Creates an anchor (links) to the same webpage or other websites
    • href - points the url for the link
      • (ex: < a href: (img url) < /a>)
